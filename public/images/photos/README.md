# Languages Cube Photo Slots

Replace `placeholder.svg` usage in `data/site.js` with uploaded WebP/JPG files after adding real photos.

Recommended filenames:

- `home-hero.webp` - strongest classroom, instructor-student, or stage image for the homepage hero
- `student-proof.webp` - student group or classroom proof image
- `about-institute.webp` - institute, founder, team, or reception photo
- `demo-form.webp` - friendly instructor consultation or student guidance photo
- `methodology-classroom.webp` - speaking practice or classroom activity photo
- `classroom-moments.webp` - gallery category image
- `stage-events.webp` - stage/event gallery category image
- `student-group.webp` - student gallery category image
- `instructor-group.webp` - instructor gallery category image
- `certificates.webp` - certificate/results gallery category image
- `workshops.webp` - workshop gallery category image
- `cultural-workshop.webp` - event card image
- `stage-presentation.webp` - event card image
- `exam-bootcamp.webp` - exam workshop image

SEO image rules:

- Use real Languages Cube photos where possible.
- Rename files descriptively before upload.
- Prefer landscape images for hero and event sections.
- Keep faces away from extreme edges so mobile crops look natural.
- Compress photos before production; WebP at 1600px wide is usually enough.
